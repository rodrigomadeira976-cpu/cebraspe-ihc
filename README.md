# Cebraspe UnB 2026 — Protótipo HTML/CSS

Este pacote contém uma simulação em HTML/CSS/JS das 8 telas enviadas:

1. Página Inicial
2. Página Vestibular UnB
3. Login / Área do Candidato
4. Dados Pessoais
5. Curso e Campus
6. Upload de Documentos
7. Revisão
8. Confirmação e Boleto

## Como abrir

1. Extraia o ZIP.
2. Abra a pasta no VS Code.
3. Clique com botão direito no `index.html`.
4. Escolha **Open with Live Server**.

Também é possível abrir o `index.html` direto no navegador.

## Arquivos

- `index.html`: estrutura das telas.
- `style.css`: aparência visual.
- `script.js`: navegação entre telas.
